# REAL 交付门禁清单（必须逐项通过）

用于防止“只完成锚点图和音乐就误判可成片”。

## Gate A：素材齐套

- [ ] 锚点图 A/B/C 已选定（含 URL）
- [ ] 视频分段素材 S1..Sn 已真实生成（每段有 URL）
- [ ] 音乐主轨已生成（audio_url）

## Gate B：可剪辑（含回归质检）

- [ ] 导演分段单已完整（含首尾帧对齐与重跑策略）
- [ ] 剪辑执行单已完整（时间线/拼接窗/字幕/导出）
- [ ] 段间拼接窗口验证通过（至少关键转场）
- [ ] 视频链路已确认为：锚点图 -> 视频参考图 -> i2v（非纯 t2v）
- [ ] 景别配比通过：大全<=30%，中景>=40%，近/特写>=30%
- [ ] 产品露出 >=2 次
- [ ] Logo 清晰露出 >=1 次，且停留 >=1.5s
- [ ] 总时长严格等于用户指定值（不可沿用默认30s）
- [ ] 节奏非机械等长，情绪曲线成立（前稳->中准/中强->后收）

## Gate C：成片交付

- [ ] 30s 主版 URL
- [ ] 15s A 版 URL
- [ ] 15s B 版 URL
- [ ] Clean/Sub 套件 URL（或明确待补）
- [ ] ffmpeg/导出通道可用并有实际导出日志
- [ ] Discord 大文件发送受限时，已先发 Review 版审片

## 抓总发布规则

仅当 Gate C 通过后，抓总才能发布：
- `REAL Master Brief 完成`
- `可发布/可交付`
