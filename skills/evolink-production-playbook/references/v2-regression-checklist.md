# v2 回归测试单（基于上一轮问题）

用于下一轮品牌营销视频执行前后的硬性检查。

## A. 流程链路
- [ ] 已完成：品牌DNA -> 母题 -> 三幕
- [ ] 已完成：锚点图（A/B/C）审美验收
- [ ] 已完成：从锚点图生成视频参考图（R-S1..R-Sn）
- [ ] 视频生成模式 = i2v（非纯t2v）

## B. 真实门禁
- [ ] S1..Sn 均有真实 `task_id`
- [ ] S1..Sn 均有真实 `video_url`
- [ ] 无将 pending 误报为 failed
- [ ] 失败段仅单段重跑

## C. 画面质量
- [ ] 非全大景：大全<=30%，中景>=40%，近/特写>=30%
- [ ] 产品露出 >= 2 次
- [ ] Logo 清晰露出 >= 1 次，且停留 >= 1.5s
- [ ] 人物一致性稳定（发型/服饰/轮廓不漂移）
- [ ] 色板稳定（黑/白/金主基调不漂移）

## D. 节奏与叙事
- [ ] 总时长严格等于用户指定值
- [ ] 段数按节奏动态设计（非机械等长）
- [ ] 情绪曲线成立：前稳 -> 中准/中强 -> 后收
- [ ] 关键秒点具备可感知节拍变化

## E. 剪辑与交付
- [ ] ffmpeg/导出通道可用
- [ ] Master30 已导出并可访问
- [ ] 15s A/B 已导出并可访问
- [ ] Clean/Sub 套件已导出（或明确缺件）
- [ ] Discord 发送策略：大文件先发 Review 版

## F. 抓总发布规则
仅当 E 项通过后，才允许发布：
- REAL完成
- 可发布/可交付
