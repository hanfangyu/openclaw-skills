# VFX Evolink Delivery

## 已验证模型（优先）

- 图像：`doubao-seedream-4.5`
- 视频：`doubao-seedance-1.0-pro-fast`
- 音乐（联动参考）：`suno-v4.5`

## 输入

1. 编剧稿（文案/分镜）
2. 导演可拍版（机位/节奏/转场）
3. 项目约束（时长、比例、质量、品牌露出）

## 输出清单

1. 风格板（关键词、色彩、材质、光效）
2. 逐镜头特效点（镜头号/秒点/特效层/风险）
3. 模板资产（HUD/粒子/数据流命名）
4. 合成质控（抠像/景深/辉光/动模糊/噪点）
5. 交付规范（工程层级/命名/导出参数）

## Evolink 执行规则

- 图像先出关键帧锚点（3-6张）再指导视频段落。
- 若视频模型单次时长不足，按 5s 段落拆分并保留 0.3-0.5s 重叠。
- 单段失败时仅重试该段；重试2次失败再切 fallback。
- 统一风格锚点：主体、色板、HUD motif 保持一致。
