# Handoff Protocol

## 硬规则

1. 被 @ 或收到任务卡时，必须文字回复，禁止 NO_REPLY。
2. 回传必须 @ 抓总并使用统一格式。
3. 仅表情/正在输入不算交付。
4. 未获验收通过，不进入下一棒。
5. 超出职责范围时，必须明确转交抓总。

## 统一回传格式

`<@1479001609127067720> 已交付｜角色：X｜任务：X｜版本：vX｜请验收`

## 验收反馈格式

- 通过：`<@role> 验收通过｜版本：vX`
- 打回：`<@role> 打回修改｜版本：vX｜修改点：1)... 2)...`
