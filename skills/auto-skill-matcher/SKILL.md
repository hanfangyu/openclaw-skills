# 自动技能匹配器
## 核心功能
1. 接收到任务需求后，首先检索本地 `/root/.openclaw/workspace/skills` 目录下所有技能的名称和描述，匹配最相关的技能
2. 本地无匹配技能时，自动调用ClawHub搜索在线可用技能
3. 支持自动安装匹配到的最优技能，无需手动操作
4. 支持阈值配置，相似度超过设定值自动安装，低于则提示用户选择
## 适用场景
所有任务执行前自动触发，无需手动调用，自动完成技能匹配和安装
## 使用方法
### 主动搜索技能
```bash
python3 matcher.py "你的任务关键词"
```
### 返回结果示例
```
✅ 本地匹配到技能：weibo-publisher（微博发布技能）
匹配度：92%
是否需要调用？
```
```
❌ 本地无匹配技能，在线搜索到：
1. skill-finder-cn（技能查找器，匹配度87%）
2. adaptive-suite（自适应工具集，匹配度76%）
输入序号自动安装
```
